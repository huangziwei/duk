# duk 讀

`duk` is essentially a Chinese counterpart to [nik](https://github.com/huangziwei/nik) and [neb](https://github.com/huangziwei/neb), it converts both Simplified and Traditional Chinese EPUBs to M4B using [Qwen3-TTS](https://github.com/QwenLM/Qwen3-TTS) (0.6B-Base, 6-bit) via [mlx-audio](https://github.com/Blaizzy/mlx-audio).
